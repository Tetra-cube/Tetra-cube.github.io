D&D 5e Content Reference

There's a lot of content for this game spread out across a bunch of different rulebooks. I made a quick reference to keep track of all the content that pertains to player characters, mostly for purposes of character creation.

Rulebooks

  • Player's Handbook is the corebook which contains all of the base content.
  • Elemental Evil Player's Companion is a short online supplement, containing a few races and some spells.
  • Sword Coast Adventurer's Guide is primarily about the Sword Coast setting, but contains new subraces, subclasses, backgrounds, and a couple of spells.
  • Volo's Guide to Monsters contains 13(!) additional races, including playable monster races.
  • Xanathar's Guide to Everything has a tremendous amount of new subclasses, as well as spells and racial feats.
  • Dungeon Master's Guide has a few options for player characters that require DM approval.
  • Other Content: Blood Hunter Class contains the Blood Hunter, the only official class outside of the Player's Handbook. The Tortle Package contains the Tortle race. Adventure Modules sometimes contain a small amount of usable content.
  • Mordenkainen's Tome of Foes is an upcoming book scheduled for release in May.
  • Unearthed Arcana is not a rulebook, but playtest material found on the Wizards of the Coast website. This content may be unbalanced, be careful when using it.
