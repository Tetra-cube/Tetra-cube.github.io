D&D 5e Content Reference

There's a lot of content for this game spread out across a bunch of different rulebooks. I made a quick reference to keep track of all the content that pertains to player characters, mostly for purposes of character creation. Also check out the random character generator, based on the same data.

Rulebooks

  • Player's Handbook is the corebook which contains all of the base content.
  • is a short online supplement, containing a few races and some spells.
  • is primarily about the Sword Coast setting, but contains new subraces, subclasses, backgrounds, and a couple of cantrips.
  • contains 13(!) additional races, including playable monster races.
  • has a tremendous amount of new subclasses, as well as spells and racial feats.
  • contains the gith race and various subraces.
  • has a few options for player characters that require DM approval.
  • The Tortle Package contains the tortle race. Adventure Modules sometimes contain a small amount of usable content.
  • contains the various races of Eberron, and dragonmarks. This is UA material.
  • is not a rulebook, but playtest material found on the Wizards of the Coast website. This content may be unbalanced, be careful when using it. See the bottom of the page for a list of Unearthed Arcana content.
  • includes popular unofficial content such as the Blood Hunter.
